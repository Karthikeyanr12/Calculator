# 🧮 Simple Calculator (HTML, CSS, JavaScript)

## 📌 Description
This project is a **simple calculator web application** built using **HTML, CSS, and JavaScript**. It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

The main goal of this project is to **practice core frontend development concepts** and understand how JavaScript interacts with the DOM.

---

## 🛠 Tech Stack
- HTML5  
- CSS3  
- JavaScript  

---

## ✨ Features
- Perform basic arithmetic operations (+, −, ×, ÷)  
- Interactive user interface  
- Responsive design  
- Clear functionality  
- Real-time input and output display  

---

## 🎯 What I Learned
- DOM manipulation using JavaScript  
- Handling user input and events  
- Styling with CSS (layout, spacing, responsiveness)  
- Structuring web pages using HTML  
- Implementing logic for calculations
